**Project Overview: Product Inventory Analysis**


This Python project demonstrates a simple inventory analysis system for a list of products. It performs the following tasks:

Calculate total value of each product – Multiply the unit price by quantity to get the total value for each product.
Check for low stock items – Identify products with quantity less than 5 and print an alert.
Accumulate total inventory value – Keep a running sum of all product values to get the total inventory worth.

This lab exercise is designed to practice Python loops, dictionary handling, arithmetic operations, and conditional statements.


**Features**
Product Value Calculation: Computes the total monetary value of each product.
Low Stock Alerts: Prints alerts for items with quantity less than 5.
Total Inventory Value: Aggregates total value for all products for quick summary. 



**Project Overview: Global Talent Auditor – Week 2 Master Lab**


The Global Talent Auditor is a Python script designed to identify high-earning employees in the Data department while handling incomplete salary data. This assignment demonstrates conditional filtering, loop logic, data validation, and basic aggregation.

**Features:**
Data Filtering: Identifies employees earning more than ₦300,000 in the Data department.
Data Validation: Skips employees with missing salary data (None) and prints a warning message.
Aggregation: Calculates the average salary of the filtered employees.
Result Tracking: Maintains a list of high earners for reporting.
Dataset
A list of dictionaries representing 10 employees.
Each dictionary contains:
name (string) – Employee name
department (string) – Employee department
salary (float or None) – Monthly salary
Usage
Run the script in Python 3.
Observe warnings for employees with missing salaries.
View the list of high earners and the average salary.

 **Output**
Data Integrity Error for Employee Enoch
Data Integrity Error for Employee Iremide

High Earners: ['Prestige', 'David', 'Grace']
Average Salary of High Earners: $442,500

**Notes:**

Uses a loop with continue for clean handling of missing salary data.
Average salary is computed only from valid, filtered entries.
Easily extendable to larger datasets or integrated with a CSV/JSON database.
